# T.P.2-Objetos-Concesionaria
Este proyecto simula el funcionamiento de una **concesionaria de vehículos** (autos y motos) utilizando **Java** con un enfoque totalmente orientado a objetos. La aplicación carga una lista fija de vehículos y, mediante el uso del **Collections Framework** y la **Stream API**, genera distintos reportes por consola.
